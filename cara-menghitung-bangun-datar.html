1. Cara menghitung bangun datar


Jawab:Banyak jenis bangun datar yang dapat kita hitung luas maupun kelilingnya, berikut beberapa rumus:

Persegi

Luas = s x s

Keliling = 4 x s

Ket :

s = sisi

Persegi Panjang

Luas = p x l

Keliling = 2 x ( p + l )

Ket :

p = panjang

l = lebar

Segitiga

Luas= 1/2 x a x t

Keliling = sisi a + sisi b + sisi c

Ket :

a = alas

t = tinggi

Jajar Genjang

Luas = a x t

Keliling = 2 x (sisi a + sisi b)

Ket :

a = alas

t = tinggi

Layang – Layang

Luas = 1/2 x d1 x d2

Keliling = 2 x (sisi a + sisi b)

Ket :

d1 = diagonal 1

d2 = diagonal 2

Belah Ketupat

Luas = 1/2 x d1 x d2

Keliling = 4 x sisi

Ket :

d1 = diagonal 1

d2 = diagonal 2

Trapesium

Luas = 1/2 x (a + c) x t

Keliling = sisi a +sisi b +sisi c +sisi d

Ket :

a = alas

c = sisi yang sejajar dengan alas

Lingkaran

Luas = π x r x r

Keliling = 2 x π x r = π x d

Ket :

r = jari – jari

c = diameter

PEMBAHASAN

Bangun datar yang merupakan panggilan untuk  bangun - bangun dua dari ukuran yang terdiri atas luas, lebar, tinggi, dll dari suatu kelompok elemen saling berhubungan yang membandingkan nilai sebuah besaran yang diukur dengan besaran semacam yang digunakan sebagai satuan, gabungan bangun datar mampu Membuat bangun ruang sehingga Bangun tersebut yang dipisahkan beberapa  ruas garis

Jenis-Jenis Bangun Datar PersegiTrapesium dibagi menjadi beberapa Jenis : Trapesium siku-siku, Trapesium sembarang, Trapesium sama kaki Persegi Panjang Jajar Genjang Segitiga dibagi menjadi beberapa Jenis :  Segitiga sembarang, Segitiga lancip, Segitiga siku-siku, Segitiga tumpul, Segitiga sama kaki, dllPersegi Panjang Lingkaran dll

Rumus Bangun Datar

Rumus Persegi :

Luas Persegi : L = s²Panjang Diagonal Persegi : d = [tex]\tt d^2 = (\sqrt{2\times s})^2[/tex]Sisi Persegi : s = [tex]\tt \sqrt{L}[/tex] Atau s = Keliling : 4 Keliling Persegi : K = 4s

Rumus Persegi Panjang

Lebar persegi panjang : l = L : p  Atau l = (Keliling : 2) – p Luas persegi panjang : L = p × lKeliling persegi panjang  : K = 2(p+l)Panjang diagonal persegi panjang : [tex]\tt d = \sqrt{p^2+l^ 2}[/tex]

Rumus Segitiga

Alas Segitiga  : a = (2 × Luas) : tKeliling Segitiga  : K = sisi × sisi × sisiTinggi Segitiga : t = (2 × Luas) : a

Rumus Jajar Genjang

a diketahui L  : a = L : tKeliling Jajar Genjang  : K = 2 × alas + 2× sisi miring Sisi Sisi Miring : b = (Keliling : 2) - aLuas Jajar Genjang  : L = Alas × tinggi

Rumus Trapesium

Keliling Trapesium  : K = Jumlah seluruh sisi Luas Trapesium : [tex]\tt \frac{1}{2} \times (a+b) \times t[/tex]Tinggi Trapesium : [tex]\tt t = \frac{2\times L}{(a+b)}[/tex]

16. Bagaimanakah cara menghitung keliling bangun datar


↪Rumus keliling dan luas bangun datar↩

Mencari keliling dan luas persegi :

• K = 4 × s

• L = s × s

Mencari keliling dan luas persegi panjang :

• K = 2 × (p + l)

• L = p × l

Mencari keliling dan luas segitiga :

• K = sisi 1 + sisi 2 + sisi 3

• L = ½ × (diagoal 1 × diagonal 2)

Mencari keliling dan luas lingkaran :

• K = 2 × π × r / π × d

• L = π × r²

Mencari keliling dan luas jajar genjang :

• K = 2 × (alas + sisi miring)

• L = alas × tinggi

Mencari keliling dan luas belah ketupat :

• K = 4 × sisi

• L = ½ × (diagonal 1 × diagonal 2)

Mencari keliling dan luas layang-layang :

• K = 2 × (sisi 1 + sisi 2)

• L = ½ × (diagonal 1 × diagonal 2)
