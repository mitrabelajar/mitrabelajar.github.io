Cara menentukan keliling bangun datar adalah dengan menjumlahkan seluruh sisinya. Jika bangun datar tersebut memiliki 3 sisi, maka rumusnya K = a + b + c. Jika memiliki 4 sisi, maka rumusnya K = a + b + c + d. Jika memiliki 5 sisi, maka rumusnya K = a + b + c + d + e, dan seterusnya.
