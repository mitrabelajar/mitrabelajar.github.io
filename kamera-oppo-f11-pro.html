10. oppo artinya apa⁉️⁉️​


Oppo artinya Apa (Bhs. Jawa).

Pembahasan

Ukara interogatif yaiku jinising ukara sing ngemot pitakon sing tujuwane kanggo njaluk wangsulan utawa wangsulan saka wong liya. Unsur-unsur 5W + 1H Bahasa Jawa.

5W + 1H

• What - Apa = Apa dibaca (Oppo).

• Why - Kenapa = Kenangapa.

• Who - Siapa = Sapa.

• When - Kapan = Kapan.

• Where - Dimana = Ing Ngendi.

• How - Bagaimana = Kepriye.

CONTOH KALIMAT

1. Apa biji ulanganmu apik?

2. Kenangapa Kowe mesem-mesem dhewek?

3. Sapa sing dadi Juri Lomba Nulis?

4. Kapan Kowe angkat Sekolah?

5. Kowe tiba ing ngendi?

6. Kepriyé kabar Ibu lan Bapakmu?

14. Jika anda melakukan kombinasi tombol Shift + F11 pada Ms. Excel maka akan…


Jika anda melakukan kombinasi tombol Shift + F11 pada Ms. Excel maka akan sama dengan menyisipkan new worksheet atau lembar kerja baru.  

Pelajari juga tentang pengertian Microsoft EXCEL, baca di https://brainly.co.id/tugas/12605835  

Pembahasan

Microsoft excel merupakan salah satu pengembangan software Microsoft Office yang berfungsi sebagai  pengoperasian angka. Dalam pengoperasiannya Ms. Excel memiliki berbagai menu utama yang memiliki fungsi tersendiri, antara lain sebagai berikut :

Menu file menu home menu insert  menu page layout menu formulas menu data menu review menu view

Tidak hanya itu, untuk mempermudah user dalam pengoperasian angka Ms. Excel juga memiliki kombinasi shortcut key untuk membantu pengguna bekerja lebih cepat.  Contoh :

Shift + F1 untuk menampilkan panel bantuan Excel. Shift + F2 untuk menambahkan atau mengedit komentar sel. Shift + F3 untuk menampilkan kotak dialog sisipkan fungsi. Shift + F4 untuk menampilkan kotak dialog  penutup jendela kerja yang dipilih. Shift + F5 untuk menampilkan kotak dialog yang di buka. Shift + F6 untuk beralih antar lembar kerja, kontrol zoom, panel tugas, dan pita. Shift + F7 untuk Membuka kotak dialog ejaan untuk memeriksa ejaan dalam lembar kerja aktif. Shift + F8 untuk menambahkan rentang sel atau sel yang tidak berdekatan ke pilihan sel menggunakan tombol panah. Shift + F9 untuk menghitung lembar kerja aktif. Shift + F10 untuk menampilkan menu Pemeriksaan kesalahan. Shift + F11 untuk menyisipkan lembar kerja baru. Shift + F12 untuk menyimpan lembar kerja yang dikerjakan.
