4. tentukan rumus jumlah suku pertama deret aritmatika adalah​


Jawab:

Sn = U₁ + U₂ + U₃ + ... + U(n - 1) + Un  => ini rumusnya

⇔ Sn = a + a + b + a + 2b + ... + a + (n - 1)b

⇔ Sn = (2a + (n - 1)b)

⇔ Sn = (a + a + (n - 1)b)

⇔ Sn = (a + Un)

S(n - 1) = U₁ + U₂ + U₃ + ... + U(n - 1)

Sn - S(n - 1) = Un

19. rumus kedua dan kelima suatu deret aritmatika masing masing 5 dan 14. rumus jumlahan n suku pertama deret aritmatika tersebut dapat dinyatakan sebagai​


Jumlah n suku pertama deret aritmatika tersebut dapat dinyatakan sebagai :

Sn = (3n² + n) / 2

=======================================

Deret aritmatika adalah penjumlahan dari suku - suku barisan bilangan yang memiliki pola yang tetap di mana polanya berdasarkan operasi penjumlahan atau pengurangan. Singkatnya, deret aritmatika menghitung jumlah barisan aritmatika.

Deret aritmatika dihitung dengan rumus Sn = n/2 (a + Un) di mana Un merupakan suku ke - n dari barisan aritmatika yang dihitung dengan Un = a + (n - 1)b.

PEMBAHASAN :

Perhatikan kembali soal di atas.

Suku kedua dan kelima suatu deret aritmatika masing - masing 5 dan 14. Maka kita akan menentukan nilai suku awal (a atau U1) dan beda suku yang berdekatan (b) dengan menguraikan kedua suku yang diketahui.

Un = a + (n - 1)b

U2 = a + (2 - 1)b

5 = a + b atau a = 5 - b .....persamaan 1

Kemudian,

U5 = a + (5 - 1)b

14 = a + 4b

a = 14 - 4b .....persamaan 2

Satukan kedua persamaan untuk menentukan nilai b.

a = a

5 - b = 14 - 4b

-9 = -3b

b = 3

Substitusikan nilai b ke persamaan 1.

a = 5 - b

a = 5 - 3

a = U1 = 2

Dengan demikian, suku ke - n (Un) deret aritmatika ini adalah Un = a + (n - 1)b

Un = 2 + (n - 1)3

Un = 2 + 3n - 3

Un = 3n - 1

Jadi, jumlah n suku pertama (Sn) deret aritmatika ini dapat dihitung dengan :

Sn = n/2 (a + Un)

Sn = n/2 (2 + (3n - 1))

Sn = n/2 (3n + 1)

Sn = (3n² + n) / 2
