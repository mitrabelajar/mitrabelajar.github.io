1. rumus dari tabung Luas tabung dan Volum tabung​


Jawaban:

luas:π×r. volume:π×r+r×t


2. rumus luas dan volum tabung?


Rumus Volume Tabung: V = π x r² x t
Dimana : V = volume tabung.
                π = phi ( 22/7 atau 3,14 ).
                r  = jari-jari alas. ( r = setengah diameter )
                t  = tinggi tabung.
Rumus luas permukaan tabung  = 2 x π x r x ( t + r )
