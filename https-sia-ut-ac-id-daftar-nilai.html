6. Jika panjang siai BC adalah 10 cm, panjang sisi AB dan AC adalah...​


Jawaban:

5 cm dan 5 akar 3 cm

Penjelasan dengan langkah-langkah:

Pada segitiga istimewa yang sudutnya 90⁰, 30⁰, 60⁰

Perbandingan sisi nya adalah (pada gambar)

AB : BC : AC = 1 : 2 : akar 3

==>

AB = 1/BC × 2 = 1/10 × 2

AB = 5 cm

AC = akar 3/BC × 2 = akar 3/10 × 2

AC = 5 akar 3 cm

16. klasifikasi teknologi informasi berikan daftar pustakanya jawab baik" jangan sia siakan koin​


- IT (Information Technology) -

1. Hardware (Perangkat Keras)

> bagian fisik dari teknologi.

Komponen utama hardware sistem komputer:

a. Unit Masukan

Alat untuk memasukkan data / perintah ke dalam komputer. Contoh : keyboard, mouse, scanner, etc.

b. Central Processing Unit

Otak dari sistem komputer, terdiri atas; Arithmatic and Logical Unit, Control Unit / CU dan Main Memory.

c. Unit Keluaran

Menampilkan/mencetak keluaran sebagai hasil pengolah data. Contoh : printer, monitor, plotter, speaker, etc.

d. Memory

Menyimpan data, informasi dan program pengolah.

| Komponen Pendukung Hardware

| I/O Ports

| Data Bus

| Address Bus

| Control Bus

2. Software (Perangkat Lunak)

> Operating System

Software yang digunakan untuk mengendalikan resources selama proses berlangsung. Fungsi sistem operasi adalah mengalokasikan memori dan pekerjaan serta penjadwalan pekerjaan. Contoh : DOS, UNIX, Windows, Linux, dsb.

> Programming Language

Digunakan untuk mengembangkan program aplikasi. Level/generasi bahasa pemrograman komputer (4 GL = 4 Generation Language) :

1. Bahasa mesin (machine language)

2. Bahasa rakitan (assembly language)

3. Bahasa pemrograman tingkat tinggi (programming language)

4. Aplikasi (application)

> Query Language

Software bahasa tingkat sangat tinggi yang dapat digunakan untuk menampilkan informasi-informasi yang diinginkan hanya dengan sedikit perintah saja. contoh : SQL (structured query language)

> Application (App)

Software yang dikembangkan untuk digunakan pada kegiatan/pekerjaan tertentu. contoh : SPSS, Ms. Office, Photoshop, dll.

3. Brainware (Perangkat Otak)

> Brainware yaitu manusia sebagai user dari Sistem Komputer. Selain itu dikenal istilah lain yaitu firmwere dan infowere. Firmwere adalah Softwere yang tertanam semipermanen di dalam suatu hardwere. Infowere adalah dokumen pendukung softwere, seperti lisensi, User Manual, Help dll.

KLASIFIKASI.

19. Jenis otorisasi dimana limit user id didaftarkan oleh sendiri oleh perusahaan disebut?


Klik BCA Bisnis adalah salah satu jenis otorisasi yang dimiliki oleh salah satu bank resmi di Indonesia yaitu Bank Central Asia atau BCA. Yang mana tentu saja cara mendaftar layanan ini harus dilakukan oleh perusahaan yang bersangkutan langsung pada bank terutama dalam peregistrasian user baru.

Ada jenis lain dari otorisasi ini yaitu Klik BCA Individu yang diperuntukkan bagi perorangan dalam menggunakannya.

Pembahasan :

Klik BCA Bisnis adalah produk digital dari Bank BCA berbasis web yang digunakan untuk melakukan transaksi dan pengontrolan keuangan untuk usaha/bisnis. Secara umum juga dikenal dengan istilah Internet Banking. Adanya internet banking ini sangat memudahkan bagi nasabah bank untuk mengakses layanan-layanan yang disediakan oleh bank. Pengguna hanya perlu mengaktifkannya dan menghubungkannya lewat berbagai berbagai perangkat, contohnya seperti handphone, tablet, iPad, dan lain sebagainya. Klik BCA Bisnis adalah jenis otorisasi bank yan sangat memudahkan nasabah bank tersebut.
