4. ciri ciri grafik eksponensial dan logaritma


Mempunyai Sifat                                                                                                Fungsi eksponensial adalah salah satu fungsi yang paling penting dalam matematika. Biasanya, fungsi ini ditulis dengan notasi exp(x) atau ex, dimana e adalah basis logaritma natural yang kira-kira sama dengan 2.71828183.Fungsi eksponensial (merah) terlihat hampir mendatar horizontal (naik secara sangat perlahan) untuk nilai x yang negatif, dan naik secara cepat untuk nilai x yang positif.Sebagai fungsi variabel bilangan real x, grafik ex selalu positif (berada di atas sumbu x) dan nilainya bertambah (dilihat dari kiri ke kanan). Grafiknya tidak menyentuh sumbu x, namun mendekati sumbu tersebut secara asimptotik. Invers dari fungsi ini, logaritma natural, atau ln(x), didefinisikan untuk nilai x yang positif.Secara umum, variabel x dapat berupa bilangan real atau bilangan kompleks, ataupun objek matematika yang lain; lihat definisi formal dibawah ini.

6. buatlah grafik fungsi eksponensial dari: 3x²​


Jawab:

Tuliskan langkah langkah menggambar grafik fungsi y = 3x²! kemudian gambarlah grafik fungsi tersebut pada kertas berpetak!

Bentuk umum fungsi kuadrat adalah f(x) = ax² + bx + c dengan a ≠ 0. Ada beberapa langkah dalam membuat grafik fungsi kuadrat. Yaitu

1. Menentukan bentuk kurva yaitu  

jika a > 0, maka kurvanya terbuka ke atas  

jika a < 0 maka kurvanya terbuka ke bawah

2. Menentukan titik potong terhadap sumbu x dan sumbu y, yaitu

Titik potong terhadap sumbu x jika y = 0

Titik potong terhadap sumbu y jika x = 0

3. Menentukan koordinat titik puncak/titik balik yaitu (xp, yp)

xp =  ⇒ biasanya disebut sumbu simetri

yp = , dengan D = b² – 4ac (D = diskriminan) atau yp = f(xp)

Pembahasan

y = 3x²

a = 3

b = 0

c = 0

Karena a > 0 yaitu a = 3, maka bentuk parabolanya terbuka ke atas

Karena b dan c = 0, maka titik potong terhadap sumbu x, titik potong terhadap sumbu y dan titik puncaknya berada di titik yang sama yaitu (0, 0)

Selanjutnya kita coba substitusikan untuk x bilangan bulat, misalkan kita mulai dari x = –3 sampai x = 3 dengan membuat  tabel

(–3, 27), (–2, 12), (–1, 3), (0, 0), (1, 3), (2, 12), (3, 27)

Arti dari definit positif adalah grafik fungsi tidak memotong sumbu X dan seluruh grafik berada di atas sumbu X.

Nah, untuk sebarang grafik eksponensial, misalkan grafik f(x), f(x) selalu lebih besar dari nol, tidak pernah bernilai nol, dan tidak pernah bernilai negatif. Dengan demikian, grafik eksponensial adalah definit positif.
