1. rumus panjang busur lingkaran???​


Panjang Busur Lingkaran

= Sudut Pusat / 360° × Keliling Lingkaran

Contoh :

Sudut Pusat = 30°

Jari - Jari Lingkaran = 7 cm

Carilah Panjang Busurnya!

Penyelesaian :

Panjang Busur

= 30° / 360° × 2πr

= 1/12 × 2 × 22/7 × 7

= 2/12 × 22

= 1/6 × 22

= 3,67 cm

5. Tuliskan rumus menghitung panjang busur lingkaran.


lingkaran merupakan bangun datar yang tersusun dari beberapa titik yang memiliki jarak yang sama terhadap titik pusat, dimana jarak antara titik pusat dengan salah satu tutuk disebut jari" lingkaran

Rumus" bab lingkaran

Luas lingkaran = π x r² atau ¹/₄ x π x d²

r = jari" lingkaran

d = diameter lingkaran

π = 22/7 atau 3,14

keliling lingkaran = 2 x π x r atau π x d

dengan d = garis tengah lingkaran

panjang busur = ∠ pusat / 360 x keliling lingkaran

Pembahasan

∠ pusat / 360 x keliling lingkaran

= ∠ pusat / 360 x 2 x π x r

ket tanda < = sudut

contoh soal:

berapa panjang busur AB jika sudut AOB = 60 derajat dan panjang jari" 7 cm

Pembahasan soal:

diket:

sudut AOB = 60

jari" = 7 cm

ditanya panjang busur AB

jawab:

panjang busur AB = ∠ pusat / 360 x 2 x π x r

                              = ⁶⁰/₃₆₀ x 2 x ²²/₇ x 7

                              = ¹/₆ x 44

                              = 7,33 cm

8. rumus mencari panjang tali busur pada lingkaran


s = r √(2(1-cosθ))


s = tali busur
r = jari-jari
θ = sudut dalam radian


Luas juring = 924 cm^, r = 42
L juring = 3.14 x r x r x sudut/360
924 = 3.14 x 42 x 42 x sudut/360
924 / (3.14 x 42 x 42) = sudut /360
924/5544 = sudut /360
0.16667 = sudut/360
sudut = 0.16667 x 360
sudut = 60'

dengan perbandingan segitiga istimewa sudut (30 , 60 , 90)
diketahui r sebagai sisi terpanjang dengan nilai perbandingan 2
dan setengah dari panjang tali busur sebagai bagian dr sisi terpendek dengan perbandingan 1

jadi 42 / 2 = 21
panjang setengah tali busur = 21
panjang tali busur = 21 x 2 = 42

solusi ke 2...
