9. rumus periode gerak melingkar


Periode (T) dalam gerak melingkar bisa di definisikan dari :

T = t/n  , n banyak putaran (dalam satuan radian) , t = waktu tempuh

atau jika diketahui kec sudut ω  , bisa dibalik dari persamaan :
ω = 2π / T  --> T = 2π / ω
ωπ/T→T=2π/ω itu adalah jawaban nya

16. apa saja rumus dari gerak melingkar dan berikan penjelasan!


Gerak Melingkar : gerak suatu benda menempuh lintasan melingkar dgn kelajuan/kecepatan tetap.
Gerak Melingkar ada 2 yaitu : 
~ GMB (Gerak Melingkar Beraturan) cirinya kec.sudut konstan, rumus : tetha = kec.sudut x waktu (Ф= ω×t)
~ GMBB (Gerak Melingkar Berubah Beraturan) cirinya perc.sudut tetap, rumus :
1. ωt = ωo + αt
2. Ф = ωo×t + 1/2αt²
3. ωt² = ωo² + 2αФ

ket : ωo : kec.sudut awal (rad/s)
ωt : kec.sudut akhir (rad/s)
α : perc. sudut (rad/s^2
Ф : posisi sudut (radian atau disingkat rad)
t : selang waktu (s)

semoga membantuuu :)Gerak melingkar  
dibagi atas 2:
1. Gerak melingkar Beraturan (GMB)
jarak tempuh,rumusnya :
s = v x t

waktu ,rumusnya:
t = s/v

kecepatan,rumusnya:
v = s/t

2. Gerak Melingkar berubah beraturan
ada 3 rumus umum :

1. Vt= Vo x α x t

2. s =  Vo x t + 1/2 x α x t²

3. Vt² = Vo²+ 2 x α x s

ket:
Vo = kec. awal
Vt = kecepatan akhir
s= jarak tempuh
α = percepatan
