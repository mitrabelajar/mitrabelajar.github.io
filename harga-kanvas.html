4. Seorang pelukis membeli 2 kanvas dan 3 cat kaleng seharga Rp70.000 suatu hari pelukis tersebut membeli 3 kanvas dan sekaleng cat di toko dan merek yang sama dengan harga Rp70.000 maka harga sekaleng cat adalah​


Jawab:

Harga sekaleng cat adalah Rp 10.000.

Penjelasan dengan langkah-langkah:

Diketahui :

Harga 2 kanvas dan 3 cat kaleng = Rp 70.000

Harga 3 kanvas dan sekaleng cat  = Rp = 70.000

Ditanyakan :

Harga sekaleng cat = ....

Penyelesaian :

Misal :

Kanvas = x

Cat = y

Maka

2x + 3y = 70.000

3x + y   = 70.000

Eliminasi x :

2x + 3y = 70.000 ⇒ x 3

3x + y   = 70.000 ⇒ x2

6x + 9y = 210.000

6x + 2y = 140.000 -

       7y = 70.000

y = 70.000 : 7

y = 10.000

Diketahui : penulis membeli 2 kanvas dan 3 cat = 70.000

3 kanvas dan 1 cat = 70.000

Ditanya : harga 1 kaleng cat (n) ?

Misalnya kanvas adalah m

Cat adalah n

Maka diperoleh model persamaan sebagai brikut

2m + 3n = 70.000

3m + 1 n = 70.000

Menggunakan metode eliminasi

2m + 3n = 70.000 |(dikali 1)

3m + n = 70.000 |(dikali 3)

Didapat

2m + 3n = 70.000

9m + 3n = 210.000 | dikurangi

Diperoleh

-7m = -140.000

m = 20.000

menggunakan substitusi

2m + 3n = 70.000

2 ( 20.000 ) + 3n = 70.000

40.000 + 3n = 70.000

3n = 30.000

n = 10.000

jadi harga 1 kaleng cat adalah 10.000

penjelasan  

1. eliminasi merupakan suatu proses yang mempunyai arti penghilangan atau pengeluaran atau bisa juga disebut penyingkiran

2. substitusi merupakan suatu proses dimana penggantian suatu hal tertentu dengan sesuatu yang baru

11. 6. Pak Amir membeli 2 lampu depan dan 3 kanvas rem dengan harga Rp 355.000,00. Pada toko yang sama Pak Sandi membeli 2 lampu depan dan 2 kanvas rem dengan harga Rp 270.000,00. Jika Pak Gun akan membeli 1 lampu depan dan 1 kanvas rem, harga yang harus dibayar oleh Pak Gun adalah


Pak amir membeli 2 lampu depan dan 3 kanvas rem dengan harga Rp. 355.000. Pak sandi membeli 2 lampu depan dan 2 kanvas rem dengan harga Rp. 270.000. Jika pak gun membeli 1 lampu depan dan 1 kanvas rem, maka pak guna harus membayar sebesar Rp. 135.000.

Penyelesaian Soal :

Misalkan harga lampu depan = x

                harga kanvas rem = y

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Pak amir membeli 2 lampu depan dan 3 kanvas rem dengan harga Rp. 355.000" yaitu :

2x + 3y = 355.000   ... (Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Pak sandi membeli 2 lampu depan dan 2 kanvas rem dengan harga Rp. 270.000" yaitu :

2x + 2y = 270.000   ... (Persamaan 2)

LANGKAH KETIGA (III)

Eliminasikan persamaan 1 dan persamaan 2 sehingga diperoleh nilai y dengan menggunakan cara sebagai berikut :

2x + 3y = 355.000

2x + 2y = 270.000

________________ -

         y = 85.000

LANGKAH KEEMPAT (IV)

Subtitusikan nilai y pada persamaan 1 sehingga diperoleh nilai x dengan menggunakan cara sebagai berikut :

2x + 3y = 355.000

2x + 3 (85.000) = 355.000

2x + 255.000 = 355.000

2x = 355.000 - 255.000

2x = 100.000

x = [tex]\frac{100.000}{2}[/tex]

x = 50.000

LANGKAH KELIMA (V)

Hitung harga yang harus dibayar pak gun dari membeli 1 lampu depan dan 1 kanvas rem dengan menggunakan cara sebagai berikut :

x + y = 50.000 + 85.000

        = 135.000

∴ Kesimpulan pak gun harus membayar 1 lampu depan dan 1 kanvas rem dengan harga Rp. 135.000.
