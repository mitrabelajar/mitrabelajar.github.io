5. tolong master mtk materi kelas 10​


Penjelasan dengan langkah-langkah:

no. 1

f(2) = 3

misal x = 2

f(x - 2) = 4 + f(x)

f(2 - 2) = 4 + f(2)

f(0) = 4 + 3

f(0) = 7

misal x = 4

f(4 - 2) = 4 + f(4)

f(2) = 4 + f(4)

3 = 4 + f(4)

f(4) = 3 - 4

f(4) = -1

misal x = 6

f(6 - 2) = 4 + f(6)

f(4) = 4 + f(6)

-1 = 4 + f(6)

f(6) = -1 - 4

f(6) = -5

bila dipasangkan

{(0, 7), (2, 3), (4, -1), (6, -5), ... (100, ..?)}

Membentuk pola

untuk domain atau x

0, 2, 3, 4, 6, ..., 100

a = U1 = 0 dan b = 2, Un = 100

tentukan banyak sukunya

Un = 100

a + (n - 1).b = 100

0 + (n - 1).2 = 100

n - 1 = 50

n = 51

untuk range atau y

7, 3, -1, ....

a = 7 dan b = -4

untuk n = 51

Un = a + (n - 1).b

U51 = 7 + (51 - 1).-4

U51 = 7 - 200

U51 = -193

Jadi, f(100) = -193

no. 2

Diket :

f(2) = -3

f(1) = 15

f(n - 1) = 2 f(n + 1) - 3 f(n)

Tentukan nilai f(4)

untuk n = 2

f(2 - 1) = 2 f(2 + 1) - 3 f(2)

f(1) = 2 f(3) - 3 f(2)

15 = 2 f(3) - 3(-3)

15 = 2 f(3) + 9

2 f(3) = 15 - 9

f(3) = 6/2

f(3) = 3

Menentukan nilai f(4)

misal n = 3

f(3 - 1) = 2 f(3 + 1) - 3 f(3)

f(2) = 2 f(4) - 3 f(3)

-3 = 2 f(4) - 3(3)

-3 = 2 f(4) - 9

2 f(4) = -3 + 9

f(4) = 6/2

f(4) = 3

Jadi, nilai dari f(4) adalah 3

17. MTK kelas 8 semester 1 materi spldv​


Jawaban:

3x + 2y = 23 ...... (1)

x - 6y = 11....... (2)

maka,

x = 6y + 11

lalu subsitusikan ke persamaan (1)

3x + 2y = 23

3(6y + 11) + 2y= 23

18y + 2y + 33 = 23

20y = 23 - 33

20y = -10

y = -½

x - 6y = 11

x -6(-½) = 11

x + 3 = 11

x = 11 - 3

x = 8

pembuktian:

3x + 2y = 23

3(8) + 2(-½) = 23

24 - 1 = 23

23 = 23

x - 6y = 11

8 - 6(-½) = 11

8 + 3 = 11

11 = 11

maka terbukti, nilai x = 8 dan y = - ½
