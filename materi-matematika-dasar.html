9. Berikan contoh SOAL DASAR tentang kombinasi dengan jawabannya !Mapel : MatematikaMateri : Kombinasi


Dari 8 orang akan dipilih 5 orang untuk menjadi tim basket sekolah, tentukan banyak cara memilih pemain tersebut!

Penyelesaian :
Diket
n = 8
r = 5
maka
n C r = n! / (n-r)! r!
8 C 5 = 8! / (8-5)! 5!
= 8 x 7 x 6 x 5! / 3 x 2 x 1 x 5!
= 8 x 7 x 6 / 6
= 56 cara

jadi ada 56 cara memilih pemain tersebut

Jawab: Siswa dapat mengenali dan memahami peran matematika di dunia. memiliki disposisl dan kapasitas untuk menggunakan pengetahuan dan keterampilan matematika untuk memecahkan masalah dalam kehidupan nyata.
