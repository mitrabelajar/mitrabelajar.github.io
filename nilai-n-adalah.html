8. quizzz n³+n⁴×n¹=nilai yang menunjukan nilai n adalah?pakai cara!!​


keterangan :

n¹ = n

n² = n x n

n³ = n x n x n

n⁴ = n x n x n x n

soal :

=> n³+ (n⁴ × n¹)

=> nilai yang menunjukan nilai n adalah? adalah?

Jawaban :

=> n³+ (n⁴ × n¹)

=> n³ + n^5

Jadi nilai yang menunjukan nilai n adalah n³ + n^5
