3. Conditional sentence tipe 1


Subject : English

Category : Structure

Sub-category : First Conditional

Level : JHS

Code : 5

Keyword : if

------------------

Conditional type 1 is used for possible situations. These situations are very likely to happen in the future when certain conditions are fulfilled. The conditions are realistic.

Structure :

If + Simple Present, Simple Future

(If + S + v₁/s, S + will + v₁)

------------------

Penjelasan:

Conditional Sentence

Conditional Sentence dibagi menjadi 3 tipe, yaitu:

Type 1, digunakan untuk mengungkapkan atau mengandaikan suatu keinginan, harapan, atau rencana yang masih mungkin bisa terjadi.Type 2, digunakan untuk mengungkapkan keinginan, harapan, atau rencana yang tidak terpenuhi, atau angan-angan yang bertentangan dengan fakta saat ini.Type 3, digunakan untuk mengandaikan sebuah peristiwa, keinginan, harapan, atau rencana yang sudah tidak mungkin lagi terjadi, karena peristiwa yang sesungguhnya sudah terjadi di masa lalu (mengandaikan peristiwa masa lalu).

Pattern

Type 1

S + will + V1 + if + S + V1 atau

If + S + V1, S + will + V1

He will study hard if he wants to pass the examination.

Type 2

S + would + V1 + if + S + V2 atau

If + S + V2, S + would + V1

He would study hard if he wanted to pass the examination.

Type 3

S + would have + V3 + if + S + had + V3 atau

If + S + had + V3, S + would have + V3

He would have studied hard, if he had wanted to pass the examination.

Note

Untuk pengandaian Type 2, yang dipakai selalu were apapun subjeknya. Tobe was tidak pernah digunakan.Type 3 bisa menggunakan Would, Could, atau Might.Type 3 bisa tanpa if (without if). Pattern : Had + S + V3 + S + Would + Have + V3. Example : Had he wanted to pass the examination, he would have studied hard.

13. 5 contoh kalimat future conditional sentence,present conditional sentence,and past conditional sentences


future/ type 3 :
If + subject + had + V3/past perfects + subject + would have + V3/past perfect

- If you had remembered to invite me, I would have attended your party.
- If he had asked you for forgiveness, would you have forgiven him?
- If he had called you, you could have gone.

CREATE ALL TYPES OF CONDITIONAL SENTENCES

"SAYA TIDAK AKAN DATANG KE RUMAHMU , JIKA HARI HUJAN "

________________________

JAWABAN :

TYPE 1

=> If it RAINS , i WILL NOT COME to your house.

TYPE 2

=> IF it RAINED , i WOULD NOT COME to your house.

TYPE 3

=> IF it HAD RAINED , i WOULD NOT HAVE COME to your house.

PENJELASAN :

CONDITIONAL SENTENCE{ KALIMAT PENGANDAIAN } :

¶ tells about the condition that will be actualized ( possible condition ) or only dreaming ( intention ) because of the particular condition.

¶ Consists -If clause and result clause.

¶ Uses Comma if the -If clause is started at the beginning sentence.

¶ Commonly, there are 4 types.

CONDITIONAL ZERO

=> is used to describe the  PHENOMENON and GENERAL TRUTH ( THE FACT )

Example :

If we boil the water, it evaporates up to atmosphere.

CONDITIONAL TYPE 1

✓ states the true in the present/future ( The planning which will be possible actualized )

✓ the speaker will actualize his/her dream.

✓ uses simple present tense and simple future

Pattern :

If +S+ verb (s/es) + O,  S+will+verb1+O/C

Example :

1. If it RAINS, i WILL NOT COME to your house

Real Condition in the present : it's possible that i don't come because it begins to rain

CONDITIONAL TYPE 1

✓ States the untrue in the present / future

✓ Contrary to the fact

✓ The speaker has intention but he/she can't afford his/her dream to be come true in the present/future

✓uses simple past and past future

✓uses Tobe ( were) for all noun / pronoun subjects ( either singular or plural )

Pattern:

If+S+verb2+O....S+Would+verb1+O/C

Example :

If it RAINED, I WOULD NOT COME to your house.

Real Condition in the present : I come to your house because It doesn't rain or it is not rainy season.

CONDITIONAL TYPE 3

✓ States the untrue in the past

✓ Contrary to the fact

✓ uses past perfect and past future perfect

Pattern :

If+S+Had+V3+O....,S+Would+have+V3

Example :

If it HAD RAINED, i WOULD NOT HAVE COME to your house

Real Condition in the Past : it didn't rain , so i came to your house

KESIMPULAN :

Conditional sentence sebagai kalimat andai-andai atau hanya impian jika persyaratan pendukung tidak tercapai.

Dalam bahasa inggris terdapat 4 kategori kalimat pengandaian .

Conditional Zero membahas tentang fakta/kejadian yang sebenarnya/fenomena alam. Bentuk strukturnya pada umumnya menggunakan simple present.

Conditional type 1 membahas tentang prediksi/rencana benar yang mungkin saja terjadi karena persyaratannya diwujudkan/direalisasikan saat ini atau mendatang. bentuk strukturnya pada umumnya menggunakan simple present dan simple future.

Conditional type 2 membahas tentang kejadian yang kontras dengan fakta nya yang terjadi saat ini atau mendatang. Dalam istilah ini, si pengucap hanya berandai-andai saja kerena persyaratannya tidak diwujudkan. Pada umumnya bentuk strukturnya menggunakan simple past dan past future tense.

Conditional type 3 membahas tentang kejadian yang kontras dengan faktanya yang terjadi di waktu lampau. Kondisi  tidak terwujud di waktu lampau.
