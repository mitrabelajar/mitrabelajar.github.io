6. 100 poin Nilai Mutlak kelas 10. ​


Bentuk sederhana [tex]|x|+|x+1|-|2-x|[/tex] untuk :

a. x ≤ -1 adalah -x-3.

b. -1 ≤ x ≤ 0 adalah x-1.

c. 0 ≤ x ≤ 2 adalah 3x-1.

d. x ≥ 2 adalah x+3.

PEMBAHASAN

Tanda mutlak adalah nilai suatu bilangan tanpa tanda plus atau minus. Contoh |2| = |-2| = 2. Pada tanda mutlak berlaku sifat sebagai berikut :

[tex]|x|=\left\{\begin{matrix}-x,~~x< 0\\ \\x,~~x\geq 0\end{matrix}\right.[/tex]

.

DIKETAHUI

Tanda mutlak [tex]|x|+|x+1|-|2-x|[/tex].

.

DITANYA

Tentukan bentuk sederhananya untuk :

a. x ≤ -1.

b. -1 ≤ x ≤ 0.

c. 0 ≤ x ≤ 2.

d. x ≥ 2.

.

PENYELESAIAN

x bernilai negatif pada saat x < 0 dan positif pada saat x ≥ 0. Sehingga |x| dapat kita tulis sebagai :

[tex]|x|=\left\{\begin{matrix}-x,~~x< 0\\ \\x,~~x\geq 0\end{matrix}\right.[/tex]

.

x+1 bernilai negatif pada saat x < -1 dan positif pada saat x ≥ -1. Sehingga |x+1| dapat kita tulis sebagai :

[tex]|x+1|=\left\{\begin{matrix}-(x+1),~~x< -1\\ \\x+1,~~x\geq -1\end{matrix}\right.[/tex]

.

2-x bernilai negatif pada saat x > 2 dan positif pada saat x ≤ 2. Sehingga |2-x| dapat kita tulis sebagai :

[tex]|2-x|=\left\{\begin{matrix}-(2-x),~~x> 2\\ \\2-x,~~x\leq 2\end{matrix}\right.[/tex]

.

a. x ≤ -1.

Pada interval ini :

x bernilai negatif

x+1 bernilai negatif

2-x bernilai positif, maka

[tex]|x|+|x+1|-|2-x|=-(x)-(x+1)-(2-x)[/tex]

[tex]|x|+|x+1|-|2-x|=-x-3[/tex]

.

b -1 ≤ x ≤ 0.

Pada interval ini :

x bernilai negatif

x+1 bernilai positif

2-x bernilai positif, maka

[tex]|x|+|x+1|-|2-x|=-(x)+(x+1)-(2-x)[/tex]

[tex]|x|+|x+1|-|2-x|=x-1[/tex]

.

c. 0 ≤ x ≤ 2.

Pada interval ini :

x bernilai positif

x+1 bernilai positif

2-x bernilai positif, maka

[tex]|x|+|x+1|-|2-x|=(x)+(x+1)-(2-x)[/tex]

[tex]|x|+|x+1|-|2-x|=3x-1[/tex]

.

d. x ≥ 2.

Pada interval ini :

x bernilai positif

x+1 bernilai positif

2-x bernilai negaitf, maka

[tex]|x|+|x+1|-|2-x|=(x)+(x+1)+(2-x)[/tex]

[tex]|x|+|x+1|-|2-x|=x+3[/tex]

.

KESIMPULAN

Bentuk sederhana [tex]|x|+|x+1|-|2-x|[/tex] untuk :

a. x ≤ -1 adalah -x-3.

b. -1 ≤ x ≤ 0 adalah x-1.

c. 0 ≤ x ≤ 2 adalah 3x-1.

d. x ≥ 2 adalah x+3.

.

15. Grafik fungsi Nilai mutlak kelas 10​


Pernyataan yang benar mengenai grafik fungsi y = 2|x - 1| + 3|x - 2| adalah memotong sumbu Y di titik (0,8) → (C).

Pembahasan

Nilai mutlak ialah jarak antar sebuah bilangan dengan 0 di garis bilangan. Nilai mutlak selalu bernilai positif, tidak ada yang negatif.

Penyelesaian

Diketahui

Grafik fungsi y = 2|x - 1| + 3|x - 2|

Ditanya

Pernyataan yang benar

JAWAB

Menentukan titik potong sumbu x, y = 0.

Menentukan titik potong sumbu y, x = 0.

→ Titik potong sumbu y, x = 0

[tex]y = 2 |x - 1| + 3 |x - 2| [/tex]

[tex]y = 2 |0 - 1| + 3 |0 - 2| [/tex]

[tex]y = 2 | - 1| + 3 | - 2| [/tex]

[tex]y = 2.1 + 3.2[/tex]

[tex]y = 2 + 6[/tex]

[tex]y = 8[/tex]

[tex] \boxed{y = (0,8)}[/tex]

Kesimpulan

Berdasarkan pembahasan di atas, pernyataan yang benar mengenai grafik fungsi y = 2|x - 1| + 3|x - 2| adalah memotong sumbu Y di titik (0,8) → (C).
