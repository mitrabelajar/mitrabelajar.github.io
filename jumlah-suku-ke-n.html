16. Jumlah suku ke n suatu barisan ditentukan dengan rumus n²+n. Nilai suku ke 8 adalah...


Penjelasan dengan langkah-langkah:

Sn = n² + n

Un = Sn - S(n - 1)

Un = (n² + n) - ((n - 1)² + (n - 1))

Un = n² + n - (n² - 2n + 1 + n - 1)

Un = n² + n - (n² - n)

Un = n² + n - n² + n

Un = 2n

U8 = 2 x 8

U8 = 16

17. jumlah n suku barisan aritmatika adalah 14 jika suku pertamanya n dan suku ke n adalah 3, tentukan n​


Jawaban:

n = 7

Penjelasan dengan langkah-langkah:

Rumus deret aritmatika, atau jumlah dari n suku barisan aritmatika adalah:

Sn = (n/2)(U1 + Un)

Dengan suku pertama = n, suku ke-n = 3, dan banyak suku = n, dapat dihitung:

14 = (n/2)(n + 3)

28 = n² + 3n

n² + 3n – 28 = 0

(n – 7)(n + 4) = 0

n = 7 atau n = –4

Karena penyelesaian yang valid hanya berlaku untuk n > 0, maka penyelesaiannya adalah;

n = 7

Jadi, suku pertama adalah 7, suku ke-n adalah 3.

Jika ingin dicari beda/selisih antar suku, bisa dihitung sbb.

Un = U1 + (n – 1)b

3 = 7 + 6b

6b = 3 – 7 = –4

b = –2/3
