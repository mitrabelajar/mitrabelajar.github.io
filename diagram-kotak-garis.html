Pembahasan

Membuat Diagram Dahan-Daun

Diagram dahan-daun (stem and leaf plot) merupakan diagram yang menyajikan data dalam bentuk dua kolom angka. Kolom pertama merupakan berlaku sebagai dahan dan kolom kedua berlaku sebagai daun.

Pada soal di atas diketahui :

Data hasil pengamatan terhadap kemampuan hidup (dalam menit) dari contoh acak 25 ekor lalat yang telah disemprot dengan insektisida adalah sebagai berikut:

2.4, 0.7, 3.9, 2.8, 1.3, 1.6, 2.9, 2.6, 3.7, 2.1, 3.2, 3.5, 1.8, 3.1, 0.3, 4.6, 0.9, 3.4, 2.3, 2.5, 0.4, 1.5, 4.3, 1.8, 2.4

a. Bentuk diagram dahan-daun.

b. Nilai kuartil 1, kuartil 2, dan kuartil 3.

Baiklah, mari kita bahas dan selesaikan satu persatu pertanyaan di atas.

a. Bentuk diagram dahan-daun

Sebelum membuat diagram dahan-daunnya, terlebih dulu kita urutkan data dari yang terkecil.

0.3, 0.4, 0.7, 0.9, 1.3, 1.5, 1.6, 1.7, 1.8, 1.8, 2.1, 2.3, 2.4, 2.4, 2.5, 2.6, 2.8, 2.9, 3.1, 3.2, 3.4, 3.5, 3.9, 4.3, 4.6

Banyak data = 25

Setelah data diurutkan, sekarang kita langsung membuat diagram dahan-daunnya. Angka sebelum koma kita anggap sebagai dahannya dan angka setelah koma kita anggap sebagai daunnya. Untuk 0.3, 0 diletakkan di bagian dahan dan 3 diletakkan di bagian daun.

Dahan Daun

0 3

Berikutnya untuk 0.4, karena 0 sudah ada maka tinggal meletakkan 4 di bagian daun sesudah angka 3.

Dahan Daun

0 3 4

Begitu seterusnya untuk 0.7 dan 0.9

Dahan Daun

0 3 4 7 9

Sekarang lanjut ke 1.3, 1 diletakkan di bagian dahan dan 3 diletakkan di bagian daun.

Dahan Daun

0 3 4 7 9

1 3

Berikutnya untuk 1.5, karena 1 sudah ada maka tinggal meletakkan 5 di bagian daun sesudah angka 3.

Dahan Daun

0 3 4 7 9

1 3 5

Begitu seterusnya sampai data terakhir.

Diagram dahan-daun selengkapnya dapat dilihat di lampiran.

b. Nilai kuartil 1, kuartil 2, dan kuartil 3.

Kuartil 1 (Q₁)

Rumus : 1/4 (n + 1), n = banyak data

Q₁ = 1/4 (25 + 1)

Q₁ = 1/4 (26)

Q₁ = 26/4

Q₁ = 6,5

Artinya kuartil 1 terletak di antara suku ke-6 dan ke-7. Berdasarkan data yang telah diurutkan pada soal a, maka data yang terletak pada suku ke-6 dan ke-7 adalah 1.5 dan 1.6, sehingga :

Q₁ = 1.5 + 1.6 / 2

Q₁ = 3.1/2

Q₁ = 1.55

Kuartil 2 (Q₂)

Rumus : 1/2 (n + 1), n = banyak data

Q₂ = 1/2 (25 + 1)

Q₂ = 1/2 (26)

Q₂ = 26/2

Q₂ = 13

Artinya kuartil 2 terletak di pada suku ke-13. Berdasarkan data yang telah diurutkan pada soal a, maka data yang terletak pada suku ke-13 adalah 2.4, sehingga :

Q₂ = 2.4

Kuartil 3 (Q₃)

Rumus : 3/4 (n + 1), n = banyak data

Q₃ = 3/4 (25 + 1)

Q₃ = 3/4 (26)

Q₃ = 78/4

Q₃ = 19,5

Artinya kuartil 3 terletak di antara suku ke-19 dan ke-20. Berdasarkan data yang telah diurutkan pada soal a, maka data yang terletak pada suku ke-19 dan ke-20 adalah 3.1 dan 3.2, sehingga :

Q₃ = 3.1 + 3.2 / 2

Q₃ = 6.3/2

Q₃ = 3.15

c.

- Bentuk diagram kotak garis

Diagram kotak garis adalah diagram berupa garis yang memuat nilai terkecil, kuartil 1, kuartil 2, kuartil 3, dan nilai terbesar sebuah data. (Bentuk diagramnya dapat dilihat pada lampiran).

- Kesimpulan tentang kesimetrikan data

Kesimpulan yang bisa diambil berdasarkan diagram di atas adalah :

1. Berat terbesar (4.6) lebih jauh ke Q₃ (2.4) daripada berat terkecil (0.3) ke Q₁ (1.55), artinya sebaran data cenderung ke kanan.

2. 25% data terletak di antara berat terkecil (0.3) dengan Q₁ (1.55), dan 25% data terletak di antara Q₃ (3.15) dengan berat maksimum (4.6).

3. Kotak memuat 50% data, namun data antara Q₁ dengan Q₂ lebih menyebar dibandingkan dengan antara Q₂ dengan Q₃.

- Apakah ada data pencilan

Yang dimaksud dengan data pencilan adalah datum yang menyimpang sangat jauh dari datum-datum lainnya di dalam satu kumpulan datum.

Untuk mengetahui data pencilan terlebih dulu harus dihitung jangkauan interkuartilnya (JK), yaitu kuartil 3 dikurang dengan kuartil 1

Q₃ - Q₁ = 3.15 - 1.55 = 1.60

Kemudian tentukan nilai pagar dalam (PD).

PD = Q₁ - 1½ (JK)

PD = 1.55 - 1½ (1.60)

PD = 1.55 - 2.4

PD = - 0.9

dan nilai pagar luar (PL).

PL = Q₃ + 1½ (JK)

PD = 3.15 + 1½ (1.60)

PD = 3.15 + 2.4

PD = 5.55

Dari hasil nilai pagar dalam dan pagar luar di atas ternyata tidak ada data yang terletak di bawah pagar dalam dan di atas pagar luar. Itu artinya tidak ada data pencilan pada data tersebut di atas.
