1. Contoh kalimat tenses be going to present (+)(-)(?) dan be going to past (+)(-)(?)...


*Present
pattern:
(+) = S + V1(s/es) + O(Adj./Adv/etc.)
(-) = S + Do/Does + not + V1 + O
(?) = Do/Does + S + V1 + O ?
Example :
(+) = He buys a car now
(-) = He does not buy a car now
(?) = Does he buy a car now ?

Note: keterangan waktu tidak harus ditulis.

*Past
pattern:
(+) = S + V2 + O
(-) = S + Did + not + V1 + O
(?) = Did + S + V1 + O ?
Example:
(+) = She went to school yesterday
(-) = She did not go to school yesterday
(?) = Did she go to school yesterday ?

Note: keterangan waktu tidak harus ditulis

15. Rumus to be going to


Kalimat positif :
Subyek + to be + going to + verb1
contoh :
I’m going to go swimming tomorrow.

Kalimat negatif
Subyek + to be + not + going to + verb1
Contoh :
He is not going to play football.

Kalimat interrogatif
To be + subyek + going to + verb1?
Contoh :
Are you going to Banyuwangi next week?

20. Which one is the correct positive pattern of Simple Future Tense (Going to) ? *a. Subject + to be + not + going to + verb 2b. Subject + to be + going to + verb 1c. Subject + going to + verb 1 + to bed. Subject + to be + not + going to + verb 1​


Answer:

B. Subject + to be + going to + verb 1

Future simple tense :

(+) S + to be (am/is/are) + going to + V1

(-) S + to be (am not/is not (isn't)/are not (aren't)) + going to + V1

(?) To be (am/is/are) + S + going to + V1?

I = am

He/she/it = is

You/they/we = are
