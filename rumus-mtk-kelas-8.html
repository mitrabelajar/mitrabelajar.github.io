5. 1.sebuah fungsi di definisikan f:2x+b jika f(2) =3 f(3) =5 tentukan rumus fungsi tersebut dan carilah f(25)2.sebuah fungsi didefinisikan h: mx+n jika h(2) =1 dan h(4) =5 tentukan rumus fungsi berikut h(31)MTKKelas 8Relasi Dan Fungsi​


1. f(x) = 2x + b

f(2) = 3

f(3) = 5

Tentukan nilai b.

Pilih salah satu dari f(2) = 3 atau f(3) = 5

(untuk menentukan nilai b)

f(2) = 3 → 2(2) + b = 3

4 + b = 3

b = 3 - 4

b = -1

atau

f(3) = 5 → 2(3) + b = 5

6 + b = 5

b = 5 - 6

b = -1

Rumus fungsi:

f(x) = 2x - 1

f(25) = 2(25) - 1

= 50 - 1

= 49

2. h(x) = mx + n

h(2) = 1 → 2m + n = 1

h(4) = 5 → 4m + n = 5

Eliminasi

2m + n = 1

4m + n = 5

_________ -

-2m = -4

m = 2

Substitusi m = 2 ke persamaan 2m + n = 1

2m + n = 1

2(2) + n = 1

4 + n = 1

n = 1 - 4

n = -3

Rumus fungsi:

h(x) = 2x - 3

h(31) = 2(31) - 3

= 62 - 3

= 59

Semoga membantu.

Jawab:

SOAL 1

Mencari Rumus fungsi f(x)

f(x) = 2x + b

f(2) = 2(2) + b = 3

f(2) = 4 + b = 3

f(2) = b = -1

f(x) = 2x - 1

Mencari Rumus Fungsi f(25)

f(x) = 2x - 1

f(25) = 2(25) - 1

f(25) = 50 - 1

f(25) = 49

SOAL 2

Mencari rumus fungsi h(x)

h(x) = mx + n

Eliminasi h(2) dan h(4)

h(x) = 2m + n = 1

h(x) = 4m + n = 5

————————- —

-2m = -4

m = 2

h(x) = 2x + n

h(2) = 2(2) + n = 1

h(2) = 4 + n = 1

h(2) = n = -3

h(x) = 2x - 3

Mencari rumus fungsi h(31)

h(x) = 2x - 3

h(31) = 2(31) - 3

h(31) = 62 - 3
