1. pelajaran kimia kelas 10​


Apabila diketahui lambang atom klorin adalah [tex]\sf Cl^{35}_{17}[/tex] . Maka konfigurasi elektron menurut sistem Bohr untuk ion klorin [tex]\sf Cl^{-}_{17}[/tex] adalah 2.8.8 (opsi B) .
---Adapun konfigurasi elekron untuk unsur klorin  [tex]\sf Cl^{35}_{17}[/tex]  adalah [Ne] 3s² 3p⁵ (opsi B). Sistem ini mengikuti konfigurasi elektron mekanika kuantum.
---Atom  [tex]\bf Cl^{35}_{17}[/tex]   dan  [tex]\bf Cl^{36}_{17}[/tex]   (opsi B) merupakan pasangan atom/unsur yang termasuk isotop. Hal ini karena jumlah proton kedua unsur sama-sama 17, tetapi nomor massa berbeda 35 dan 36.
---Pasangan atom atau unsur  [tex]\bf K^{39}_{19}[/tex]  dan   [tex]\bf Ca^{40}_{20}[/tex]  (opsi E) termasuk isoton. Hal ini karena atom K dan Ca sama-sama memiliki neutron sebanyak 20.

Penjelasan dengan langkah-langkah

Konfigurasi elektron menunjukkan urutan, susunan, dan/atau posisi penempatan elektron-eletron yang dimiliki suatu atom/unsur pada kulit atom atau unsur tersebut. Di sini akan dipaparkan singkat terkait dua model konfigurasi elektron yang sering digunakan, yaitu model Bohr dari Neils Bohr dan mekanika kuantum.

Penentuan konfiguasi elektron model Bohr :

Tentukan jumlah elektron unsur/atom yang diketahui.Mulai penempatan elektron dari kulit dekat inti sampai menjauhi inti dengan ketentuan :
Jumlah elektron maksimum = [tex]\bf 2n^2[/tex]
Dengan, n = nomor kulit.

Penentuan konfigurasi elektron model mekanika kuantum :

Tentukan jumlah elektron unsur/atom yang diketahui.Mulai penenmpatan elektron dari sub-level yang memiliki energi paling rendah yaitu 1 s² dan seterusnya sesuai dengan prinsip Aufbau.
---
Sublevel energi terendah selalu sublevel 1 s, yang terdiri dari satu orbital. Elektron tunggal atom hidrogen akan menempati orbital 1s ketika atom dalam kondisi dasarnya. Saat melanjutkan dengan atom dengan elektron lebih banyak, maka elektron tersebut ditambahkan ke sublevel terendah berikutnya: 2s, 2p, 3s, dan seterusnya.
---

Nomor 2

Menentukan elektron dari atom Cl
Jumlah elektron = nomor atom - muatan
Jumlah elektron = 17 - (0) = 17Gunakan pintasan jumlah elektron dari unsur gas yang terdekat, yaitu Ne karena jumlah elektronnya 10 sedangkan Cl adalah 17 (masih di bawahnya 7 elektron).Membuat konfigurasi dari 7 elektron sisa
[Ne] = 1s² 2s² 2p⁶
Adapun sisanya = 3s² 3p⁵
Konfigurasinya : [Ne] 3s² 3p⁵ (Opsi B)

Nomor 3

Isotop = atom-atom dengan jumlah proton sama.Proton diihart dari angka di bawah lambang unsur :
Opsi A, C, D, dan E memiliki nomor atom berbeda.Maka opsi B adalah pasangan isotop.
Jumlah proton kedua unsur Cl pada opsi B = 17.

Nomor 4

Isoton = atom-atom dengan jumlah proton sama.Proton diihart dari angka di bawah lambang unsur :
Opsi A, B, C, dan D memiliki jumlah neutron berbeda.Maka opsi E adalah pasangan isoton :
Jumlah neutron = nomor massa - nomor atom
Pada opsi E :
Jumlah neutron K = 39 - 19 = 20.
Jumlah neutron Ca = 40 - 20 = 20

Sehingga dengan memasukkan (mensubstitusikan) nilai-nilai ini ke rumus tadi, maka diperoleh penyelesaian berikut ini. x = \frac{-3\pm\sqrt{3^2-4.2.1}}{2.2} x = \frac{-3\pm\sqrt{9-8}}{ 4} = \frac{-3\pm 1}{4} Jadi penyelesaiannya yaitu x = -1 atau x = -\frac{1}{2}

1. aFe + bO2 -> cFeO

2Fe + O2 -> 2FeO



2.
aKClO3 -> bKCl + cO2
K : a = b
Cl : a = b
O : 3a = 2c
misal a = 2, maka b = 2, c = 3
jadi,
2KClO3 -> 2KCl + 3O2



3.
aC2H6 + bO2 -> cCO2 + dH2O
koefisien :
C : 2a = c
H : 6a = 2d
O : 2b = 2c + d
misal a = 2, maka b = 7, c = 4, d = 6
jadi,
2C2H6 + 7O2 -> 4CO2 + 6H2O



4.
aNaOH + bH2SO4 -> cNa2SO4 + dH2O
koefisien :
Na : a = 2c
O : a + 4b = 4c + d
H : a + 2b = 2d
S : b = c
misal a = 2, maka b = 1, c = 1, d = 2
jadi,
2NaOH + H2SO4 -> Na2SO4 + 2H2O



5.
aCa(OH)2 + bHCl -> cCaCl2 + dH2O
koefisien :
Ca : a = c
O : 2a = d
H : 2a + b = 2d
Cl : b = 2c
misal a = 1, maka b = 2, c = 1, d = 2
jadi,
Ca(OH)2 + 2HCl -> CaCl2 + 2H2O

13. Bantu jawab dong nomor 1-3kelas 10 IPA ,,pelajaran kimia​


1. Diketahui : Ar O = 16 g/mol; bilangan Avogadro = 6,02 x 10^23; massa gas O = 4 gram (jumlah partikel dalam 4 gram gas oksigen tsb?)

Jawab :

Mr O2 = (2 x Ar O) -> semua unsur gas selalu berdampingan (O2)

           = 2 x 16 g/mol

           = 32 g/mol

mol O2 = massa O2 / Mr O2

             = 4 gram / 32 g/mol

             = 0,125 mol

jumlah partikel O2 = mol O2 x bil. Avogadro

                               = 0,125 x 6,02 x 10^23

                               = 0,7525 x 10^23 atau 7,525 x 10^22 (A)

2. Diketahui : jumlah atom unsur X = 3,01 x 10^22; massa unsur X = 3 gram (Ar unsur X?)

Jawab :

jumlah atom unsur X = mol unsur X x bil. Avogadro

3,01 x 10^22 = mol unsur X x 6,02 x 10^23

mol unsur X = 0,5 x10^-1 atau 0,05 mol

mol unsur X = massa X / Ar X

0,05 mol = 3 gram / Ar X

Ar X = 60 g/mol (D)

3. Diketahui : massa gas O2 = 8 gram; Ar O = 16 g/mol (volume O2 pada keadaan RTP?)

Jawab :

mol O2 = massa O2 / Mr O2

            = 8 gram / 32 g/mol

            = 0,25 mol

volume O2 pada RTP = mol O2 x 24,4 L

(24,4 liter adalah volume untuk 1 mol suatu unsur pada keadaan RTP)

                                    = 0,25 mol x 24,4 L

                                    = 6,1 L atau 6 liter (A)

Pembahasan

Tata nama senyawa adalah serangkaian sistem pemberian nama yang diberikan pada molekul-molekul kimia berdasarkan pedoman IUPAC (International Union of Pure and Applied Chemistry).

Tata Nama Senyawa Ionik

Cara memberi nama untuk senyawa ionik adalah dengan:

Menggabungkan nama ion positif dan ion negatif.Tidak menggunakan awalan mono-, di-, tri-, dst.).Ion positif (unsur logam) diletakkan di kata pertama dan ditambah akhiran -ium.Ion negatif (unsur nonlogam) diletakkan di kata kedua dan ditambah akhiran -ida.

Contoh: CaCl2 (Kalsium klorida).

Tata Nama Senyawa Kovalen

Cara memberi nama untuk senyawa kovalen (ikatan non logam dengan non logam) adalah:

Nama terdiri atas dua kata.Setiap kata diberi awalan mono, di, tri, tetra, penta, dan seterusnya sesuai dengan jumlah atom.Khusus kata pertama, tidak perlu menggunakan awalan mono.Kata kedua diberi akhiran -ida.

Contoh: CCl4 (karbon tetraklorida).

Tata Nama Senyawa Ionik pada Logam Golongan Transisi

Harus mencantumkan bilangan oksidasi dengan menggunakan angka romawi dalam tanda kurung.Tidak menggunakan awalan mono-, di-, tri- , dst).Nama logamnya ditulis dalam bahasa Indonesia.

Contoh: FeO, Besi(II) Oksida.
