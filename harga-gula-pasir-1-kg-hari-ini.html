DIKETAHUI :Pada saat harga gula pasir Rp 10.000,- per kg jumlah gula pasir di Toko A yang diminta sebanyak 300 kg per , dan ketika harga gula pasir berubah Rp 12.000,- per kg jumlah gula pasir yang diminta 250 kg per hari. Sebaliknya jika harga gula pasir Rp 11.500 per kgpenjual/produsen Toko A mau menjual sebanyak 350 kg per hari, dan jika harga gula pasir Rp 13.000 per kg penjual mau menjual sebanyak 450 kg per hari. Selain itu terhadap gula pasir yang akan diperjual belikan diberikan subsidi oleh pemerintah sebesar Rp 2250.- per kg. DITANYAKAN 1. Tentukan fungsi permintaan dan fungsi penawaran gula pasir.2. Tentukan harga keseimbangan gula pasir (EP) dan jumlah keseimbangan gula pasir (EQ)terjadi di pasar dan buat kurva keseimbangan pasar tersebut (keseimbangan harga sebelum subsidi).3. Hitung jumlah keseimbangan (EQs) dan harga keseimbangan (EPt) setelah diberikan subsidi oleh pemerinta​


Penjelasan:

• Harga Rp 10.000 per kg permintaan 300 kg per hari

• Harga Rp 12.000 per kg permintaan 250 kg per hari

• Harga Rp 11.500 per kg penjualan 350 kg per hari

• Harga Rp 13.000 per kg penjualan 450 kg per hari

• Subsidi oleh pemerintah sebesar Rp 2.250.- per kg

1)

Dik : Pd¹ = 10.000, Qd¹ = 300, Pd² = 12.000, Qd² = 250, Ps¹ = 11.500, Qs¹ = 350, Ps² = 13.000, Qs² = 450, t = 2.250

P - Pd¹ / Pd² - Pd¹ = Qd - Qd¹ / Qd² - Qd¹

(P - 10.000) / (12.000 - 10.000) = (Q - 300) / (250 - 300)

(P - 10.000) / 2.000 = (Q - 300) / -50

2.000Qd - 600.000 = -50P + 500.000

2.000Qd = -50P + 1.100.000 ... dibagi 2000

Qd = -1/40P + 550

Maka Fungsi Permintaan => Qd = -1/40P + 550

P - Ps¹ / Ps² - Ps¹ = Q - Qs¹ / Qs² - Qs¹

(P - 11.500) / (13.000 - 11.500) = (Qs - 350) / (450 - 350)

(P - 11.500) / 1.500 = (Qs - 350) / 50

1.500Qs - 525.000 = 50P - 575.000

1.500Qs = 50P - 50.000 ... dibagi 1.500

Qs = 1/30P + 100/3

Maka Fungsi Penawaran => Qs = 1/30P + 100/3

2. Kesetimbangan, syarat Qd = Qs

-1/40P + 550 = 1/30P + 100/3

-1/40P - 1/30P = 100/3 - 550

-3/120P - 4/120P = 100/3 - 1650/3

-7/120P = -1550/3

-21P = -186000

P = 62000/7

Qd = -1/40P + 550

Qd = -1/40 (62000/7) + 550

Qd = -62000/280 + 550

Qd = 1550/7 + 550

Qd = 1550/7 + 3850/7

Qd = 5400/7

Harga dan jumlah keseimbangan sebelum subsidi (5400/7, 62000/7)

3.

Keseimbangan Pasar Sesudah subsidi :

Qd = -1/40P + 550 => -1/40P = Qd - 550

P = -40Qd + 22000

Qs = 1/30P + 100/3 => 1/30P = Qs + 100/3

P = 30Qs + 1000

Sesudah subsidi, fungsi penawaran menjadi

P = 30Q + 1000 + 2250

P = 30Q + 3250

-40Q + 22000 = 30Q + 3250

-40 - 30Q = 3250 - 22000

-70Q = -18750

Q = 1875/7

P = 30Q + 3250

P = 30(1875/7) + 3250

P = 56250/7 + 3250

P = 56250/7 + 22750/7

P = 79000/7

Harga dan jumlah keseimbangan setelah subsidi (1875/7, 79000/7)
