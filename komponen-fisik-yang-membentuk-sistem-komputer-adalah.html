Jawaban singkat:

Hardware

Penjelasan tambahan:

Secara garis besar, komputer terdiri dari 3 perangkat. Yaitu:

→ Perangkat keras (hardware) yaitu komponen fisik

→ Perangkat lunak (software) yaitu komponen non-fisik

→ Perangkat otak (brainware) yaitu manusia selaku pemakai komputer

Berdasarkan fungsinya, hardware terdiri dari 4 jenis. Yaitu:

1. hardware input : contohnya keyboard

2. hardware output: contohnya monitor

3. hardware pemroses: contohnya prosessor

4. hardware penyimpanan: contohnya harddisk

PembahasanKeyboard, mouse, webcam, dan mikrofon merupakan beberapa contoh perangkat keras (hardware) untuk meng-input data ke dalam komputer. Alat-alat tersebut disebut dengan input hardware atau perangkat input (masukan).Jenis perangkat keras lain sebagai media penyimpanan yang dulu banyak digunakan adalah floppy disk (disket). Saat ini banyak pengguna yang menggunakan media bernama SSD (Solid State Drive) untuk menyimpan data digital.ROM memiliki kepanjangan Read Only Memory yang artinya adalah memori hanya untuk membaca. Apabila data disimpan ke dalam ROM, berarti data tersebut tidak dapat disunting (edit) maupun dihapus. Penggunaan ROM biasanya untuk menyimpan data yang tidak perlu dilakukan perubahan banyak dan sering, seperti firmware.Type writer merupakan bagian yang terdapat pada keyboard yang mendominasi. Istilah ini digunakan karena mengadopsi pendahulunya yaitu mesin ketik. Dalam mesin ketik hanya terdapat tombol-tombol huruf, angka, dan juga tanda baca.Terdapat 4 macam perangkat keras komputer, yaitu: input hardware (masukan), output hardware (keluaran), processing hardware (pengolah), dan juga storage media (media penyimpan).
