Penyelesaian :

Ada beberapa hal yg perlu diingat dlm mengerjakan soal untuk materi FPB dan KPK.

Ciri-ciri untuk soal KPK adalah : terdapat kata "kapan akan bertemu kembali" dan "mereka bertemu 5 hari sekali".

Ciri-ciri soal FPB adalah : memuat kata "banyaknya" atau "jumlah".

Jadi dilihat dari ciri-ciri soal KPK dan FPB, soal tsbt diatas termasuk ke dalam soal FPB.

Cara mencari FPB ada dua cara yaitu dengan menggunakan pohon faktor dan menggunakan tabel.


Menggunakan cara tabel

60, 12, 48

---------------- 2

30, 6, 24

---------------- 2

15, 3, 12

---------------- 3

5, 1, 4


FPB dari 60, 12 dan 48 = 2² x 3 = 12


Jadi jumlah bingkisan yg bisa dibuat adalah 12 bingkisan


Menggunakan pohon faktor

60 = 2 x 2 x 3 x 5

12 = 2 x 2 x 3

48 = 2 x 2 x 3 x 2 x 2

-------------------------------

FPB = 2 x 2 x 3 = 12


Jadi jumlah bingkisan yg bisa dibuat adalah 12 bingkisan

Bab Bangun Ruang
Matematika SMP Kelas VIII

1a] tabung tanpa tutup
    r = d/2
      = 10/2
      = 5 cm

   luas kertas = π x r x r + (2 x π x r x t)
                     = 3,14 x 5 x 5 + (2 x 3,14 x 5 x 25)
                     = 78,5 + 785
                     = 863,5 cm²
  
1b] t = 20 cm
 luas permukaan = π x r x r + (2 x π x r x t)
                           = 3,14 x 5 x 5 + (2 x 3,14 x 5 x 20)
                           = 78,5 + 628 
                           = 706,5 cm²

  perbandingannya = Luas permukaan 1 : luas permukaan 2
                               = 863,5 : 706,5
                               = 1.613 : 1.413

2] volume air yang tumpah = volume bola
                                           = (4/3) x π x r³
                                           = (4/3) x 22/7 x (7/2)³
                                           = 179,67 cm³
