6. jawaban buku matematika kelas 7 semester 2 hal 173 no 10​


Hubungan sudut yang besarnya 105° dan sudut yang besarnya 3b° berpelurus, sedangkan sudut yang besarnya 3b° dan sudut yang besarnya 3a° luar berseberangkan. Jadi, nilai a + b pada gambar adalah 50°.

Penjelasan dengan langkah-langkah:

Diketahui:
Sudut yang besarnya 105° berpelurus dengan sudut yang besarnya 3b°.
Sudut yang besarnya 3b° luar berseberangan dengan sudut yang besarnya 3a°
Ditanyakan:
Berapa nilai a + b?
Jawab:
Sudut yang saling berpelurus jumlah besar sudutnya adalah 180°, sehingga:
105° + 3b° = 180°
3b° = 180° - 105°
3b° = 75°
b° = 25°

Sudut yang luar berseberangan memiliki besar sudut yang sama, sehingga:
3a° = 3b°
3a° = 3(25°)
3a° = 75°
a° = 25°

Menghitung nilai a + b:
a + b = 25° + 25°
a + b = 50°
Dengan demikian, nilai a + b = 50°.

3²ˣ ⁺ ¹ + 8 . 3ˣ - 3 > 0, HP = ?

_______________________________________

JAWABAN:

HP = { x | x > -1 }Opsi (B) x > -1

PENJELASAN:

⇔ Eksponen adalah bentuk bilangan berpangkat atas suatu bilangan basis yang berarti bilangan basis dikalikan dengan bilangan basis sebanyak pangkatnya.

⇔ Sifat-sifat eksponen yang digunakan:

aᵇ ⁺ ⁿ = aᵇ . aⁿ(aᵇ)ⁿ = aᵇ * ⁿaᵇⁿ = (aⁿ)ᵇ

PEMBAHASAN SOAL:

⇒ 3²ˣ ⁺ ¹ + 8 . 3ˣ - 3 > 0

Selesaikan menggunakan sifat eksponen:

3²ˣ ⁺ ¹ + 8 . 3ˣ - 3 > 03²ˣ . 3 + 8 . 3ˣ - 3 > 0(3ˣ)² . 3 + 8 . 3ˣ - 3 > 0

Substitusikan n = 3ˣ:

n² . 3 + 8n - 3 > 03n² +  9n - n - 3 > 03n(n + 3) - (n + 3) > 0(n + 3)(3n - 1) > 0n + 3 = 0 V 3n - 1 = 0n + 3 - 3 = 0 - 3 V 3n - 1 + 1 = 0 + 1n = -3 V 3n : 3 = 1 : 3n = -3 V n = 1/3

Pisahkan menjadi a > 0, b > 0 dan a < 0, b < 0:

n > -3n > 1/3HP = { n > 1/3 }n < -3n < 1/3HP = { n < -3 }HP = { n > 1/3 V n < -3 }

Masukkan nilai n = 3ˣ:

n = { n > 1/3 V n < 3 }3ˣ = { n > 1/3 V n < 3 }3ˣ < -33ˣ > 1/3

Selesaikan pertidaksamaan pertama:

3ˣ < -3Fungsi pemangkatan selalu positif, jika tidak maka penyelesaian bukan bilangan real.x ∉ R

Selesaikan pertidaksamaan kedua:

3ˣ > 1/33ˣ > 3⁻¹x > -1

Diperoleh:

x > -1, makaHP = { x | x > -1 }
