8. Berikan rumus Luas permukaan bangun ruang dan volume bangun ruang




Rumus Volume, Luas Permukaan Bangun Ruang

a.     Kubus
Rumus:
·         Luas permukaan: 6 x s2 =6s2
·       Volume: s x s x s= s3
s = rusuk
    b.     Balok

Rumus
·         Luas permukaan : 2{(p x l)+(p x t)+(l x t)}
·         Volume: p x l x t
    c.      Limas
Rumus:
·         Luas permukaan: La + jumlah luas segitiga pada bidang tegak
·         Volume : 1/3 x La x t
La=luas alas
t= tinggi
    d.     Prisma
Rumus:
·         Luas permukaan : (2 x La)+(K x t)
·         Volume: La x t
La= luas alas
K= keliling alas
t= tinggi
   e.     Tabung
   Rumus:
·         Luas permukaan: 2 π r (r+t)
·         Luas selimut: 2 π r t
·         Volume : π r2 t
π= 22/7 atu 3,14
r= jari-jari alas
t= tinggi tabung
   f.       Kerucut
Rumus:
·         Luas permukaan: π r (r+s)
·         Luas selimut: π r s
·         Volume: 1/3 π r2 t
r= jari-jari lingkaran alas
s= panjang garis pelukis kerucut
t= tinggi kerucut
   g.     Bola
Rumus :
·         Luas permukaan: 4 π r2
·         Volume: 4/3 π r3
r= jari-jari bola

KUBUS

Lp= 6s^2

V= s^3

BALOK

Lp= 2(pl + pt + lt)

V= p x l x t

LIMAS SEGIEMPAT

Lp= p alas^2 + 4(p alas x t segitiga/2)

V= 1/3 L alas x t

KERUCUT

Lp= pi x r x (r + s)

V= 1/3 pi x r x r x t

BOLA

Lp= 4 pi x r x r

V= 4/3 pi x r x r x r

TABUNG

Lp= 2 pi x r x r + 2 pi x r x t

V= pi x r x r x t
