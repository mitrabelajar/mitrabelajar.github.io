a + b = 120 ...substitusi mnjadi a = 120 - b masukkan ke
50.000a + 25.000b = 4.000.000
50.000 ( 120 - b ) + 25.000b = 4.000.000
6.000.000 - 50.000b + 25.000b =4.000.000
-25.000b = 4.000.000 - 6.000.000
-25.000b = -2.000.000
b = -2000.000/-25.000
b = 80....masukkan ke
a + b = 120
a + 80 = 120
a = 120 - 80
a = 40
baju A = 40 buah
baju B = 80 buah

penjualan baju A = 40 x 57.000 = 2.280.000
penjualan baju B = 80 x 30.000 = 2.400.000
total penjualan adalah
= 2.280.000 + 2.400.000
= 4.680.000

modal baju A = 50.000 x 40 = 2.000.000
modal baju B = 25.000 x 80 = 2.000.000

keuntungan baju A = 2.280.000 - 2.000.000 = 280.000
keuntungan baju B = 2.400.000 - 2.000.000 = 400.000
total keuntungan = 280.000 + 400.000 = 680.000

baju lengan pendek

baju lengan panjang dan pendek

[tex] = \frac{38}{10} + \frac{11}{4} [/tex]
