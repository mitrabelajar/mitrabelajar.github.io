Urutan kegiatan ekonomi secara mendasar adalah sebagai berikut:

modal atau biaya (pengeluaran) ⇒ pendapatan (pemasukan)pembelian ⇒ penjualan

Impas, tidak untung maupun rugi

(d). Pengeluaran = Rp 1.500.000 dan pemasukan Rp 1.550.000

8. jawaban buku matematika paket kelas 8 semester 1 hal 30​


1. Banyak bola pada pola ke-n adalah 4n-3

2. Banyak bola pada pola ke-n adalah 2n² - 2n + 1

3. Jumlah bilangan pada garis ke-n pola bilangan Pascal tersebut adalah 2ⁿ⁻¹

Pembahasan

#1

Diketahui :

Pola bilangan 1,5,9,13

Ditanya :

suku ke-n

Jawab :

1, 5, 9, 13

 \/ \/  \/

 4  4  4

Perhatikan bahwa pola bilangan tersebut memiliki selisih antarsuku berurutan (beda) yang tetap  maka pola bilangan tersebut merupakan barisan aritmatika dan berlaku rumus :

Un = a + (n - 1) b

Un = 1 + (n - 1 ) 4

Un = 1 + 4n - 4

Un = 4n - 3

Jadi banyak bola pada pola ke- n adalah 4n - 3.

#2

Diketahui :

Pola bilangan 1,5,13,25

Ditanya :

suku ke-n

Jawab :

1, 5, 13, 25  --> a = 1

\/   \/   \/

 4  8  12  --> b = 4

  \/  \/

   4  4  --> c = 4

Perhatikan bahwa pola bilangan tersebut barisan aritmatika tingkat 2 sehingga  berlaku rumus :

Un = a + (n - 1) b + ¹/₂ x (n - 1) x (n - 2) x c

Un = 1 + (n - 1 ) 4 + ¹/₂ x (n - 1) x (n - 2) x 4

Un = 1 + (4n - 4) + 2 (n² - 3 n + 2)

Un= 1 + 4n - 4 + 2n² - 6n + 4

Un = 2n² - 2n + 1

Jadi banyak bola pada pola ke-n adalah 2n² - 2n + 1.

Un = 4n - 3

Jadi banyak bola pada pola ke- n adalah 4n - 3.

Ditanya :

6. Tentukan panjang AB dari gambar berikut

Perhatikan gambar yang terdapat pada lampiran II

Gambar a.

Diketahui :

AD = CD = 4 cm

BC = 3 cm

Panjang AB

AB² = (AD - BC)² + CD²

       = (4 - 3)² + 4²

       = 1 + 16

       = 17

AB = √17

Jadi panjang AB gambar a adalah √17 cm

Gambar b

Diketahui :

CD = 4 cm

BC = 7 cm

AD = 6 cm

Panjang BD pada segitiga BCD

BD² = BC² + CD²

       = 7² + 4²

       = 49 + 16

       = 65

BD = √65

Panjang AB pada Δ ABD

AB² = BD² - AD²

      = (√65)² - 6²

      = 65 - 36

      = 29

AB = √29

Jadi panjang AB gambar b adalah √29 cm

Gambar c

BO = 5 cm

       = 4² + 5²

       = 41

AB = √41

Jadi panjang AB gambar c adalah √41 cm

Ada 4 garis diagonal yang terdiri dari 4 segitiga siku-siku 

Keempat diagonal tersebut kita masukan kedalam rumus pythagoras yaitu c² = a² + b², maka PA² = a² + b²

PD² = b² + c²

 c² = 8² - d²

18. jawaban buku paket matematika kelas 8 hal 31 semester genap​


Jawaban buku paket matematika kelas 8 hal 31 semester genap. Soal yang dimaksud adalah soal “Ayo kita berlatih 6.3” tentang triple Pythagoras, yang terdiri dari 9 soal uraian. Disini saya akan menjawab soal nomor 1 sampai 5, untuk jawaban soal lainnya dapat dilihat di link pelajari lebih lanjut

Pembahasan

1. Jenis segitiga

a) 13, 9, 11

13² = 169 9² + 11² = 81 + 121 = 202

Karena 13² < 9² + 11², maka merupakan segitiga lancip

b) 8, 17, 15

17² = 289 8² + 15² = 64 + 225 = 289

Karena 17² = 8² + 15², maka merupakan segitiga siku-siku

c) 130, 120, 50

130² = 16.900 120² + 50² = 14.400 + 2.500 = 16.900

Karena 130² = 120² + 50² maka merupakan segitiga siku-siku

d) 12, 16, 5

16² = 256 12² + 5² = 144 + 25 = 169

Karena 16² > 12² + 5² maka merupakan segitiga tumpul

e) 10, 20, 24

24² = 576 10² + 20² = 100 + 400 = 500

Karena 24² > 10² + 20² maka merupakan segitiga tumpul

f) 18, 22, 12

22² = 484 18² + 12² = 324 + 144 = 468

Karena 22² > 18² + 12² maka merupakan segitiga tumpul

g) 1,73; 2,23; 1,41

2,23² = 4,9729 1,73² + 1,41² = 2,9929 + 1,9881 = 4,981

Karena 2,23² < 1,73² + 1,41² maka merupakan segitiga lancip

h) 12, 36, 35

36² = 1.296 12² + 35² = 144 + 1.225 = 1.369

Karena 36² < 12² + 35² maka merupakan segitiga lancip

2. Triple Pythagoras

a) 10, 12, 14

14² = 196 10² + 12² = 100 + 144 = 244

Karena 14² ≠ 10² + 12² maka 10, 12, 14 bukan triple Pythagoras

b) 7, 13, 11

13² = 169 7² + 11² = 49 + 121 = 170

Karena 13² ≠ 7² + 11² maka 7, 13, 11 bukan triple Pythagoras

c) 6, 2 ½, 6 ½  

(6 ½)² = (6,5)² = 42,25 6² + (2 ½)² = 6² + (2,5)² = 36 + 6,25 = 42,25

Karena (6 ½)² = 6² + (2 ½)² maka 6, 2 ½, 6 ½ merupakan triple Pythagoras
