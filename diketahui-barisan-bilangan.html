16. Diketahui Barisan Bilangan 1,3,6,10,15,21,…,…,…Barisan Bilangan ke 206. *​


Jawaban :

21.321

Langkah penyelesaian :

Pola diatas adalah pola bilangan segitiga.

Rumus pola bilangan segitiga

Un = n(n + 1)/2

Maka, pola ke-206 adalah

= 206(206 + 1)/2

= 206(207)/2

= 42.642/2

= 21.321 ✔

Keterangan :

Materi = Pola Bilangan
