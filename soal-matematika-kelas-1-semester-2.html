7. Soal MATEMATIKA KELAS XI SEMESTER 2 f(x) = x³+6x²+9x+1 Tentukan - interval naik/turun - jenis stasioner - titik stasioner - nilai stasioner.


f(x) = x³+6x²+9x+1

f'(x) = 0 = 3x²+12x+9

0 = x²+4x+3

0 = (x+3)(x+1)

x = -3 atau -1

f(-3) = (-3)³+6(-3)²-9.3+1

      = -27+54-27+1

f(-3) = 1

f(-1) = (-1)³+6(-1)²-9.1+1

     = -1+6-9+1

f(-1) = -3

nilai maksimum : 1, nilai minimum = -3, titik maks : (-3,1), titik minimum : (-1,-3)

interval turun : (x+3)(x+1) < 0 => -3 < x < -1

interval naik : (x+3)(x+1) > 0 => x < -3 atau x > -1

16. bisa bantu untuk soal matematika kelas x semester 1 edisi revisi 2014 kurikulum 2013 untuk soal uji kompetensi 1.1 no.2 b dan c makasih...


Kelas         : X SMA
Pelajaran   : Matematika
Kategori     : Eksponen dan Logaritma
Kata Kunci : Uji kompetensi 1. 1, eksponen, K-2013

Penjelasan : 

Soal ini ada pada buku paket matematika kelas X SMA K-2013 halaman 16.

Sebelum menjawab pertanyaan kita pahami sifat-sifat Pemangkatan (Eksponen)

Sifat 1
Jika x bilangan real, a dan b bilangan bulat positif maka xᵃ × xᵇ = xᵃ⁺ᵇ
Sifat 2
Jika x bilangan real dan x ≠ 0, a dan b bilangan bulat positif, maka xᵃ/xᵇ = xᵃ⁻ᵇ

Sifat 3
Jika x bilangan real dan x ≠ 0, a dan b bilangan bulat positif, maka (xᵃ)ᵇ = xᵃᵇ

Sifat 4
Jika x bilangan real dan x ≠ 0, a dan b bilangan bulat positif, maka
(xᵃyᵇ)ⁿ = xᵃⁿ . yᵇⁿ

Sifat 5
Jika x bilangan real dan x ≠ 0, dalam bentuk bilangan pecahan, maka
1 / (xⁿ) = x⁻ⁿ

Perintah soal no 2a dan 2c :

2. Dengan menggunakan sifat bilangan berpangkat, sederhanakanlah bentuk berikut.

a.  2x³ × 7x⁴ × (3x)²

c. y⁵ × (x × y)³ × [ 1 / (x² × y)]

Jawab:

a.  2x³ × 7x⁴ × (3x)² = 2x³ × 7x⁴ × (3x)(3x)
                                = 2x³ × 7x⁴ × 9x²
                                = 2 × 7 × 9 × x³⁺⁴⁺²
                                = 126 x⁹

c. y⁵ × (x × y)³ × [ 1 / (x² × y)] = y⁵ × x³ × y³ × (x² × y)⁻¹
                                               = y⁵ × x³ × y³ × x⁻² × y⁻¹
                                               = x³ × x⁻² × y⁵ × y³ × y⁻¹
                                               = x⁽³⁺⁽⁻²⁾⁾ y⁽⁵⁺³⁺⁽⁻¹⁾⁾
                                               = x y⁷
