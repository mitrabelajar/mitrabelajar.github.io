10. rumus luas dan keliling bangun datar​


1. Rumus luas persegi, yaitu:

L = S x S

Rumus keliling persegi, yaitu:

K = S + S + S + S ataupun K = 4 x S

2. Rumus luas persegi panjang, yaitu:

L = p x l

Rumus keliling persegi panjang, yaitu:

K = 2 x (p + l)

3. Rumus luas segitiga yaitu:

Luas = ½ x a x t

Rumus keliling segitiga yaitu:

Keliling = s + s + s atau K = a + b + c

4. Rumus yang ada pada Bangun Datar Jajar genjang

Nama Rumus
Keliling (Kll) Kll = 2 × (a + b)
Luas (L) L = a × t
Sisi Alas (a) a = (Kll ÷ 2) – b
Sisi Sisi Miring (b) a = (Kll ÷ 2) – a
t diketahui L t = L ÷ a
a diketahui L a = L ÷ t

5. Trapesium

• Rumus Luas Trapesium Siku-Siku
Luas trapesium (L) = ½ x jumlah panjang sisi yang sejajar x tinggi
= ½ x (AB+CD) x t

• Rumus Luas Trapesium Sama Kaki
Luas trapesium (L) = ½ x jumlah panjang sisi yang sejajar x tinggi
= ½ x (AB+CD) x t

• Rumus Luas Trapesium Sembarang
Luas trapesium (L) = ½ x jumlah panjang sisi yang sejajar x tinggi
= ½ x (AB+CD) x t

6. Rumus yang ada pada Bangun Datar Layang – layang

Nama Rumus
Luas (L) L = ½ × d1 × d2
Keliling (Kll) Kll = a + b + c + d
Kll = 2 × (a + c)
Diagonal 1 (d1) d1 = 2 × L ÷ d2
Diagonal 2 (d2) d2 = 2 × L ÷ d1
a atau b a = (½ × Kll) – c
c atau d c = (½ × Kll) – a

7. Rumus yang ada pada Bangun Datar Belah Ketupat

Nama Rumus
Keliling (Kll) Kll = s + s + s + s
Kll = s × 4
Luas (L) L = ½ × d1 × d2
Sisi (s) s = Kll ÷ 4
Diagonal 1 (d1) d1 = 2 × L ÷ d2
Diagonal 2 (d2) d2 = 2 × L ÷ d1

8. Rumus bangun datar lingkaran

Diameter (d) d = 2 × r
Jari-jari (r) r = d ÷ 2
Luas (L) L = π x r x r
atau
L = π x r2
Keliling (Kll) Kll = π x d
Mencari r r = kll/ 2π
r = √L/ √π
