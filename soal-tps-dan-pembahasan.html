14. Mohon bantuannya untuk soal TPS​


Misal, bilangan pada kolom pertama = A dan bilangan pada kolom kedua = B, maka pola operasi aritmetiknya adalah:

A*(A - 1) - B

Sehingga, untuk baris pertama dan ketiga akan didapatkan:

5*(5 - 1) - 3 = 5*4 - 3 = 20 - 3 = 7

7*(7 - 1) - 2 = 7*6 - 2 = 42 - 2 = 40

Sehingga, untuk baris kedua, akan didapatkan:

6*(6 - 1) - x = 25

6*5 - x = 25

30 - x = 25

x = 30 - 25 = 5

16. mohon bantuannya ini soal tps​


Jawaban:

8

Penjelasan:

[tex]x\sqrt{y}+y\sqrt{x} -\sqrt{7x}-\sqrt{7y}+\sqrt{7xy} = 7\\x\sqrt{y}+y\sqrt{x}+ \sqrt{7xy}=7+\sqrt{7x}+\sqrt{7y}\\\sqrt{xy}(\sqrt{x}+\sqrt{y}+\sqrt{7}) = \sqrt{7}(\sqrt{x}+\sqrt{y}+\sqrt{7})\\\sqrt{xy}=\sqrt{7} \\xy = 7[/tex]

Karena hasil kali dua bilangan = 7 dan bilangan harus ASLI. Maka kemungkinan x = 1 , y = 7 atau sebaliknya.

Sehingga x + y = 1 + 7 = 8

18. Ini gimana yah soal tps HOTS


Jawaban: 0

Penjelasan:

x faktorial atau x! = x.(x-1)! = x.(x-1).(x-2)! dst

[tex]\frac{x!}{(x+2)!} = \frac{x}{(x+1)!}[/tex]

[tex]\frac{x.(x-1)!}{(x+2)(x+2-1)!}=\frac{x}{(x+1)!}[/tex]

[tex]\frac{(x-1)!}{(x+2)(x+1)!} =\frac{1}{(x+1)!}[/tex]

(x+2) = (x-1)!

karena x adalah bilangan bulat positif, maka

x > 0 jadi hasil (x-1)! adalah 0

(x+2) = 0
